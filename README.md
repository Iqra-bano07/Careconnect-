CareConnect is a mini healthcare support web application designed to help patients connect with volunteers or NGOs for basic medical assistance.

## Features
- Simple and user-friendly interface
- Patient support request form
- Clean and responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## AI / Automation Concept
The application includes an AI-based concept where patient messages can be automatically categorized into urgent and non-urgent requests. This helps healthcare volunteers prioritize critical cases efficiently.

## Use Case
CareConnect can be used by healthcare NGOs or community volunteers to collect patient requests and provide timely support, especially for people who need quick guidance or help.
